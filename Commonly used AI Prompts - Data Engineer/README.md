# AI Prompt Library

A personal collection of reusable AI prompts, organized by area. Each file is a set of
copy-paste templates. Replace anything in `[square brackets]` with your specifics before
sending.

## How to use
1. Find the file for your task.
2. Copy the prompt block.
3. Fill in the `[placeholders]`.
4. Paste into your AI assistant. Add context/files where it helps.

> Tip: The best results come from giving the AI **role + context + task + format**.
> Most prompts here already follow that shape.
